安装

首先先安装 hexo

npm install -g hexo

下载 demo

git clone git@github.com:polarcloud/polardoc-demo.git

安装依赖并运行

cd polardoc-demo
npm install
hexo -s

然后就可以直接访问 http://localhost:4000 看到 demo 了。
